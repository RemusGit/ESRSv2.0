15. UPDATED REVERB ENV FILE SETTING -> QUEUE_CONNECTION=redis (INSTANT BROADCAST)
	- REQUIREMENTS
		- install Redis-x64-5.0.14.1.exe (C:\Redis)
		- download and copy php_redis.dll into C:\xampp\php\ext -> dl source https://repos.zend.com/zendphp/windows/archive/
		- open php.ini add this -> extension=redis
		- restart apache / php artisan optimize:clear
		- php artisan queue:work redis --sleep=1 --tries=3 --timeout=30 --max-jobs=0 --max-time=0
			sleep=0 -> instant broadcast / tries=1 prevent dups event / timeout=30 skip bad jobs / max-job=1000 prevent memory leak / max-time=3600 restart every 1hour
